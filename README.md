# Emerson's Game

A simple SNES-style 2-D platformer scaffold using `pygame`.

## Requirements
- Python 3.11+
- `pygame` 2.5+

## Development

### Install
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### Run
```bash
make run
```

### Tests
```bash
make test
```
